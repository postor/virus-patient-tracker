# virus-patient-tracker

追踪病毒传播的方案

[English version](https://github.com/postor/virus-patient-tracker/tree/english_version)

## devlopment

启动微服务

```
cross-env ELASTIC_SEARCH=your_elastic npm run dev
```

启动 webpack 并监听

```
npm run build:web:dev 
```


## production 

```
docker-compose up -d
```
